# Kata Solutions

This repository contains solutions to various coding challenges from [Codewars](https://www.codewars.com) and other platforms, organized by programming language.

## Structure

The repository is organized by programming language, with each kata in its own subdirectory:

```
.
├── CSharp/
│   └── [kata-name]/
├── PHP/
│   └── [kata-name]/
├── CPlusPlus/
│   └── [kata-name]/
├── Python/
│   └── [kata-name]/
└── TypeScript/
    └── [kata-name]/
```

## Languages

- **C#** - .NET solutions
- **PHP** - PHP solutions
- **C++** - C++ solutions
- **Python** - Python solutions
- **TypeScript** - TypeScript/JavaScript solutions

## Kata Format

Each kata directory typically contains:
- Solution file(s)
- Test file(s) (if applicable)
- README or description of the problem

## Getting Started

Navigate to the specific language directory to find kata solutions. Each kata subdirectory contains the implementation and any relevant tests.

## Contributing
