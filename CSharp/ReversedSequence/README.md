# Reversed Sequence

**Difficulty:** 8 kyu  
**Link:** https://www.codewars.com/kata/5a00e05cc374cb34d100000d

## Task

Build a function that returns an array of integers from n to 1 where `n > 0`.

## Example

```
n = 5  -->  [5, 4, 3, 2, 1]
```

## How to Compile and Test

```bash
cd CSharp/ReversedSequence
dotnet run
```

This will compile and run the tests automatically.

## Files

- `Program.cs` - Your implementation and test runner
- `ReversedSequence.csproj` - .NET project configuration
